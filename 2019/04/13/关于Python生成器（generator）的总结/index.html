关于Python生成器（generator）的总结

Posted on 2019-04-13 | In python

在自己探究scrapy爬虫框架的时候，发现有的地方看起来很是吃力，尤其是的分页的网址进行返回的一块，不是太明白，想了想，还是自己的基础知识不够全面，这才又去看了廖雪峰大神的Python教程，看到了生成器这一章节的时候，明白了许多的东西，现在绝对就generator这个方面进行一个总结，希望可以有所收获。

概述

generator：英[ˈdʒenəreɪtə(r)] 美[ˈdʒɛnəˌretɚ]
作为名词的含义有：发电机，发生器; 电力公司; 生产者，创始者; [乐] 基础低音;
确实，对于Python来说，它的确算得上是一个美妙的音符，它的灵活使用，使得Python所书写的程序更加的灵活与美妙。
它的主要应用场景集中在：

  • 获取一个定式的输出数据，但不占用内存，可以随时调用，获取下一个输出值
  • 通过generator实现程序的异步处理，使得程序更加的灵活多变
